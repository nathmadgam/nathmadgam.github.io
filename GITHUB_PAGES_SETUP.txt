CYNEX GITHUB PAGES SETUP
========================

1. Open your GitHub repository.
2. Delete the previous website files.
3. Upload every file and folder from this project to the REPOSITORY ROOT.
   The .github folder must be included.
4. Commit the files to main or master.
5. Go to Settings -> Pages.
6. Set Source to GitHub Actions.
7. Open the Actions tab and wait for:
   "Deploy Cynex portfolio to GitHub Pages"
8. After the green checkmark appears, open the URL shown in the deploy job.

For your main personal site, use a repository named exactly:
nathmadgam.github.io

Do not choose "Deploy from a branch" for this Next.js source project.
GitHub Pages must run the included GitHub Actions workflow so Next.js can
build the static out/ directory first.
