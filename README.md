# Cynex | Roblox Scripter Portfolio

This repository is a direct-upload GitHub Pages website built with plain HTML, CSS, and JavaScript.

## Publish at nathmadgam.github.io

1. Upload every file and folder in this project to the root of the `nathmadgam.github.io` repository.
2. Confirm `index.html` is visible at the repository root.
3. Open **Settings → Pages**.
4. Select either **GitHub Actions** or **Deploy from a branch → main → /(root)**.
5. Wait for deployment, then hard-refresh the website.

No npm installation, framework build, Node.js server, Prisma database, or environment variable is required.

## Main folders

- `assets/` — styles, JavaScript, posters, cached platform media, icons, and contract page previews
- `images/` — portrait media
- `videos/` — project demonstration videos
- `downloads/` — the downloadable Contract Agreement Form
- `.github/workflows/` — optional static GitHub Pages deployment workflow

The contact form prepares an email addressed to `nathanielmadridgaminde@proton.me`; it does not store visitor information.
