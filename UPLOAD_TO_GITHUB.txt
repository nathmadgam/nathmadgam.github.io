CYNEX PORTFOLIO — GITHUB PAGES ROOT UPLOAD

1. Open the repository named exactly: nathmadgam.github.io
2. Delete the old website files, especially old Next.js files and any old index.html.
3. Upload the CONTENTS of this folder directly to the repository root.
   index.html must be visible at the top level of the repository.
4. Commit the changes to main.
5. In Settings > Pages, choose either:
   - GitHub Actions (the included workflow deploys the static site), or
   - Deploy from a branch > main > /(root)
6. Wait for the Pages deployment to finish, then hard-refresh the website.

No npm install, build command, Node.js server, database, or Prisma setup is required.
